## Setup
First thing clone the repository
```
$ git clone https://github.com/mcostaca/django_api
$ cd django_api
```

Create a virtual environment
```
$ python -m venv env
$ .env/bin/activate
```

Then install the dependencies
```
(env) pip install -r requirements.txt
```
